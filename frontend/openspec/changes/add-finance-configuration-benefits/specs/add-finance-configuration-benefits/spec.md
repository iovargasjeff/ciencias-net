# add-finance-configuration-benefits Specification

## Purpose

Dar a Yanina control de configuración y beneficios futuros.

## ADDED Requirements

### Requirement 1

Cuenta financiera SHALL configurar valores futuros

#### Scenario: muestra nueva versión

- GIVEN Yanina edita periodo
- WHEN confirma
- THEN muestra nueva versión

### Requirement 2

Formulario SHALL explicar alcance y acumulación

#### Scenario: preview muestra aplicación

- GIVEN Yanina crea beneficio
- WHEN selecciona conceptos/vigencia
- THEN preview muestra aplicación
